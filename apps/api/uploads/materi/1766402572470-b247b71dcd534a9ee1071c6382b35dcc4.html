Pengenalan Formatting Shortcut

Panduan Cepat Microsoft Word

Kuasai dasar-dasar pengolah kata paling populer di dunia untuk meningkatkan produktivitas Anda.

1. Mengenal Antarmuka

Ribbon

Kumpulan tab (Home, Insert, Layout) yang berisi alat untuk mengedit dokumen.

Quick Access Toolbar

Tombol cepat untuk menyimpan (Save), membatalkan (Undo), dan mengulang (Redo).

Status Bar

Informasi jumlah halaman, kata, dan pengaturan zoom di bagian bawah.

[Ilustrasi Antarmuka Microsoft Word]

2. Dasar Formatting Teks

B

Font Style

Gunakan Bold untuk menebalkan, Italic untuk miring, dan Underline untuk garis bawah.

Alignment

Atur posisi teks: Rata Kiri, Tengah, Rata Kanan, atau Rata Kiri-Kanan (Justify).

Line Spacing

Mengatur jarak antar baris kalimat agar dokumen lebih nyaman dibaca.

3. Pintasan Keyboard Wajib

Kombinasi Tombol Fungsi
Ctrl + S Menyimpan Dokumen (Save)
Ctrl + C Menyalin Teks (Copy)
Ctrl + V Menempel Teks (Paste)
Ctrl + Z Membatalkan Perintah Terakhir (Undo)
